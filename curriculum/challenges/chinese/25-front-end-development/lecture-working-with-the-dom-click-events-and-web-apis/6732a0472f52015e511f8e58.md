什么是 API，什么是 Web API？

让我们来了解一下 API 以及它们为何对 Web 开发至关重要。API 是应用程序编程接口（Application Programming Interface）的缩写。API 建立了一套规则和协议，使软件应用能够彼此通信并高效地交换数据。

你可以把它们看作是允许开发者基于已经实现的简单构建块创建更复杂功能的结构。

API 有多种类型。Web API 是专为 Web 应用设计的。对于客户端 JavaScript 开发，有不同类型的 Web API。它们并不是 JavaScript 本身的一部分。

这些 API 通常分为两大类：浏览器 API 和第三方 API。

浏览器 API 暴露来自浏览器的数据。作为 Web 开发者，你可以使用 JavaScript 访问和操作这些数据。

它们还提供了访问各种功能的能力，比如操作网站结构、处理事件、使用存储、与网络通信等。

一些常用的浏览器 API 示例包括：

DOM API，你可以用它来操作 HTML 元素、样式和属性。在接下来的课程中你会学到更多关于 DOM 的内容。它是 Web 开发的核心概念。

Storage API，用于在用户设备上本地存储数据。

还有更多。

你可以使用各种浏览器 API 来处理用户交互，获取浏览器的更多信息，比如用户当前位置。

你还可以操作其他类型的元素，比如视频和音频。

浏览器 API 是构建强大 Web 应用的有用工具。它们已经内置在浏览器中。

但你也可以使用第三方 API，这些 API 默认并未内置在浏览器中。要使用它们，你需要以某种方式获取它们的代码。通常，它们会有详细的文档说明如何使用其服务。

一个例子是 Google 地图 API，你可以用它在网站上显示交互式地图。

第三方 Web API 有很多类型，包括天气 API、社交媒体 API、支付 API、数据 API、翻译 API 等。你只需找到适合你需求的 API。

Web API 是构建能够与各种服务和数据源交互的现代 Web 应用的重要工具。学会如何利用它们的强大功能，你将扩展自己创建动态和有吸引力网站的工具集。
